### Prerequisites

To run the image you need to have docker installed.

Get docker - https://docs.docker.com/get-docker/

## Getting Started

***Whole World*** `docker run --rm  p0bailey/corona-tools corona`

By countries: 

***Italy*** `docker run --rm  p0bailey/corona-tools corona it`
